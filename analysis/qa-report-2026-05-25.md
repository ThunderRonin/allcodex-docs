## Failures Found And Fixed

### RAG e2e Mock Env

`test/e2e/rag.e2e.test.ts` initially failed because the e2e mock env omitted the new RAG env fields.

Impact:

- Missing `RAG_HYBRID_RRF_K` produced `NaN` scores that serialized to `null`.
- Missing rerank settings produced invalid rerank payloads.

Fix:

- Added all seven RAG env fields to `test/helpers/e2e-mock-setup.ts`.
- Disabled rerank in that e2e mock path.

### Brain Dump Final SSE Event

Real brain-dump live integration initially failed because the browser did not receive the final SSE `done` event.

Impact:

- AllKnower created history and notes, but the Portal never reached `Processing Complete`.

Fix:

- Added AllKnower SSE keepalive comments during long quiet phases.
- Flushed the Portal client parser's final buffer on stream close.
